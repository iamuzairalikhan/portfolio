Fridge Setting

This interactive game challenges players to organize items in a fridge efficiently. Players open the fridge, retrieve a bucket, and strategically place items into designated fridge buckets using grid systems. Each action is scored with a star system based on the accuracy of item placement, and players can earn rewards by watching ads.

My Contributions

  • Designed and implemented grid systems within the fridge buckets to allow precise item placement and ensure organized gameplay.
  • Developed mechanics for bucket retrieval and placement, enabling seamless transitions and intuitive touch-based controls.
  • Implemented a performance-based star system to score item placement accuracy and enhance player engagement.
  • Integrated ads strategically to offer players rewards, encouraging interaction while maintaining monetization.
  • Optimized the interaction system for item movement between buckets and ensured a fluid user experience.

Key Features

  • Players can open fridge doors, retrieve buckets, and organize items interactively.
  • Grid systems in buckets help players arrange items neatly and efficiently.
  • Evaluate user organizational skills with a star system for accuracy.
  • Watch ads to earn exciting in-game rewards, adding to the incentive for perfection.
  • Multiple fridge buckets offer increasing levels of difficulty as gameplay progresses.
